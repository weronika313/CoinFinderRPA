# Coin Finder RPA
## a rpa script for scrape informations about coins from page **[metalmarket.eu](https://www.metalmarket.eu/)**


## Technologies used

* **[Python3](https://www.python.org/downloads/)** - A programming language
* **[Beautiful Soup](https://pypi.org/project/beautifulsoup4/)** - A library that makes it easy to scrape information from web pages.
* **[RPA for Python](https://pypi.org/project/rpa/)** - A simple and powerful API to makes robotic process automation

## Running and installation

   * clone repository from github

    $ git clone https://github.com/weronikazochowiec/CoinFinderRPA
   
   * create a virtualenv
   
   * install requriments
    
    $ pip install requirements.txt
    
   * run
   
    $ python main.py
   
   * After that script should generate csv file "Coins.csv" with information about silver one ounce coins
